Method

Production-first. Tool-agnostic. Clear decision-making.

A simple set of principles that keeps VP legible for production.

1) Start with Constraints

Schedule, budget, risk, locations, cast – before choosing any technique.

2) Choose the Minimum Viable Complexity

If plates solve it, use plates. If LED helps, use LED. If neither helps, don't force it.

3) Keep Production in the Loop

No black-box workflows. Decisions should be explainable to producers and line producers.

4) Protect Creative Freedom

VP should remove constraints, not add new ones.

Plates-First Thinking

We use plates heavily because they're often the quickest path to believable worlds – and they keep the process grounded in production realities.
